== Description ==

= Easily Setup Landing Page =

Landing page for WooCommerce Categories & Tags plugin allow you to add Rich text for your WooCommerce category and tag pages. It improve your keyword for SEO. No need to edit your theme or plugin files!

= Features of Landing page for WooCommerce Categories & Tags =

* Quick to setup
* Increase SEO
* You can add HTML conent and design as per your theme design

== Installation ==

1. Install Landing page for WooCommerce Categories & Tags by uploading the `landing-page-for-wc-categories-tags` directory to the `/wp-content/plugins/` directory.
2. Activate Landing page for WooCommerce Categories & Tags through the `Plugins` menu in WordPress.
3. Create Landing page for Categories & Tags by going to the `Settings > Landing Page for WC Categories & Tags` menu.
